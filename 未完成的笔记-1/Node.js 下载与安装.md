
#### Node.js 下载与安装

(以下网址可能需要科学上网)

##### 下载
* 官网：*https://nodejs.org*
* 下载地址：*https://nodejs.org/en/download/*
* 下载文件：*node-v8.9.1.tar.gz*

##### 官方文档
* 官方文档：*https://nodejs.org/en/docs/*

##### 操作系统
* CentOS Linux release 7.2

```
wget -c https://nodejs.org/dist/v8.9.1/node-v8.9.1.tar.gz

tar -zxvf node-v8.9.1.tar.gz
cd node-v8.9.1
./configure
make && make install
```
